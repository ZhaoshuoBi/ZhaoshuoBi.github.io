SOAP PUMP REDESIGN

GOAL: To build a better soap pump   FIELD: Product Design, User Research   ROLE: Team Leader
TAG: Design class group project   DURATION: Two weeks  





BACKGROUND
The act of obtaining soap to wash one’s hands is a typical activity with the assumption that others can use soap pumps with no errors. However, small cues such as frequent pumping when the soap is running low in the container clue in the user that there exists problems with using the soap pump.

Analysis
This project ends as a prototype and doesn't have a physical product. One reason is that some of our redesign feature requires a whole new structure of soap dispenser pump (the 'c' part of final prototype), which we were not capable to build; another reason is that this project was a two week design class project and mainly focus on basic design concepts and techniques.
